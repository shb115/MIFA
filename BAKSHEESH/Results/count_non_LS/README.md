# BAKSHEESH/Results/count_non_LS/ — intentionally empty

The `BAKSHEESH/Codes/count_non_LS/` scripts (`0x4_diff_check.py`,
`0x8_diff_check.py`, `all_diff_check_2r.py`, `monte_diff.py`) are
**not in the paper**; they are an artifact-only demonstration that
MIFA's non-LS analysis (DEFAULT Sec. 3.2) carries over to BAKSHEESH's
S-box / bit-perm pair.  All four scripts print to stdout only and are
**not driven by `./reproduce.sh`**.

No committed outputs live under this directory by design — running
the scripts by hand is the only way to see their numbers.  See the
top-level `README.md` (section "BAKSHEESH Sec. 2") and the scripts
themselves for the per-script commentary.
